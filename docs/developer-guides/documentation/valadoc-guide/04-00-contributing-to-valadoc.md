# 4. Contributing to Valadoc

## 4.1. Valadoc Guide (What you're reading now!)

One way to help is by improving this guide. Feel free to update the
pages in this guide by pressing the edit button on the bottom of pages
in the guide.

You can also add your own pages to this guide too.

## 4.2. Valadoc Command Line Tool

The source files for `valadoc` are inside the
[Vala Compiler Repository](https://gitlab.gnome.org/GNOME/vala).

The two key directories for `valadoc` are `libvaladoc/` and `valadoc/`.
Any help with issues and merge requests are appreciated!

## 4.3. Valadoc.org (API References Website)

[https://valadoc.org](https://valadoc.org),
is generated by combining documentation generated
by valadoc from various sources. You can learn more about how this
process works and how you can help contribute to the project in its
[git repository](https://github.com/vala-lang/valadoc-org).

## 4.4. Package sources on valadoc.org

The packages that feed the public API reference are maintained in
several upstream repositories:

-   [Vala](https://gitlab.gnome.org/GNOME/vala) (the compiler repository
    itself) — Core VAPIs and the `valadoc` sources described above
-   [vala-girs](https://github.com/nemequ/vala-girs) — GIR packages and
    related bindings metadata
-   [vala-extra-vapis](https://gitlab.gnome.org/GNOME/vala-extra-vapis)
    — Additional VAPI files maintained with GNOME Vala
