# Genie Tutorials, Blogs and Code Examples

This page provides links to tutorials, blog posts and code examples of the
Genie programming language. It has been ported from the now-retired
[GNOME Wiki page](https://wiki.gnome.org/Projects/Genie/TutorialsBlogsExamples).

Many of these resources are several years old. They may still be useful but
some links can be outdated or point to archived versions of long-gone
websites.

::: tip Help wanted
If you know of a useful resource (or a more up-to-date replacement for one
of the links below),
[open an issue](https://github.com/vala-lang/vala-docs/issues) or a pull
request on the [Vala docs repository](https://github.com/vala-lang/vala-docs).
:::

## Tutorials and Articles

- [The Genie Programming Language Tutorial](introduction) - the official
  tutorial, ported into this documentation.
  the up-to-date reference manual maintained alongside the Vala compiler.
- [Cross Compiling Vala/Genie from Linux to Android](https://github.com/avalanche-games/avalanche/wiki/Getting-Started-%28for-Linux-users%29) -
  part of the Avalanche Games project; may be interesting for advanced users.
- [Genie Doc (Español)](https://geniedoc.blogspot.com/p/home.html) - "Wiki de
  programación con Genie", an excellent set of tutorials and examples with
  screenshots and code samples; covers the basics and also GTK+3, GStreamer,
  Pango, Cairo and more.
- [Genie = Python + Vala (Polski)](http://sprae.jogger.pl/2009/07/14/genie-python-vala/) -
  some good introductory examples of using Gtk and Cairo.
- [The Genie language](http://bkhome.org/genie/) - by Barry Kauler; links to
  more detailed tutorials toward the bottom of the page.
- [Genie (programming language)](https://en.wikipedia.org/wiki/Genie_%28programming_language%29) -
  Wikipedia article.
- [GTK GUI development in Genie](https://wiki.gnome.org/Projects/Genie/GtkGuiTutorial) -
  a basic introduction from 2010.
- [In the Bottle - Good performance and easy coding](http://nnc3.com/LinuxMag/Magazine/Archive/2010/118/056-060_valagenie/article.html) -
  by Ankur Kumar, article published in 2010 in Linux Magazine.
- [Newest 'genie' Questions on Stack Overflow](https://stackoverflow.com/questions/tagged/genie) -
  technical questions and answers.
- [Raspberry Pi Home Server - Install and test the Vala language / Genie](https://www.pihomeserver.fr/en/2014/05/10/raspberry-pi-home-server-installer-et-tester-le-langage-genie/) -
  a tutorial using Genie and the WiringPi GPIO interface library to blink a
  LED on the Raspberry Pi.
- [Vala and Genie Programming - PuppyLinux](http://puppylinux.org/wikka/GenieProgramming) -
  a basic introduction from the Puppy Linux project.

## Code Examples and Utilities

- [AgenDie](https://launchpad.net/agendie) - an address book with a GTK+3
  front end.
- [GameFoo](https://github.com/darkoverlordofdata/game-foo) - "A POC for an
  ECS game engine written in Vala/Genie".
- [Genie sample code](/genie/sample-code/) — full sources hosted in this documentation (basic I/O, Cairo, GIO, LibSoup, XML, and more).
- [Image viewer](http://onagat.hatenablog.com/entry/2015/06/07/001236) -
  example of an image viewer written in Genie.
- [Image viewer 2 (GtkStack)](http://onagat.hatenablog.com/entry/2015/06/19/124943) -
  another example of an image viewer written in Genie.
- [`indent/genie.vim`](https://www.vim.org/scripts/script.php?script_id=2349) -
  Vim text editor indent script for the Genie language.
- [Khövsgöl Source Code](https://github.com/tliron/khovsgol/) - a shared music
  player and library / playlist manager licensed under the GPL. A good
  example of using GTK, SQLite and GStreamer 1.0 in Genie. Provides
  integration with PulseAudio, JACK, IceCast, RTP, desktop, MPRIS and
  Zeroconf/Bonjour.
- [ShmupWars - Vala](https://github.com/darkoverlordofdata/shmupwarz-vala) -
  arcade-style game mostly written in Genie and some Vala, uses SDL 2.
- [`py2gs`](https://github.com/tobimensch/py2gs) - a simple Bash script as a
  starting point to convert Python to Genie.
- [SDL 2.0 Example](https://github.com/sdl2-vapi/sdl2-vapi/tree/cb4df42eabdbf53702a178e048648d9ceaff14eb/examples/Example%202-%20Genie) -
  example of Genie code with the SDL 2.0 bindings. Makes use of a font,
  image and audio file.
