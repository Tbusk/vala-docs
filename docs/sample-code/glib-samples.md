# GLib Samples

These examples use core GLib APIs: date and time, legacy I/O channels, built-in
containers, markup parsing, dynamic modules, [`GLib.Value`](https://valadoc.org/glib-2.0/GLib.Value.html),
[`GLib.TypeModule`](https://valadoc.org/gobject-2.0/GLib.TypeModule.html), the GLib test harness, and
threading primitives.
